Abstracting ConfigParser as a Dictionary

When we are parsing the config file which has only one section, to get a particular config in a file we have to specify config.get(‘default’, ‘key1’). This looks little cumbersome if you have to do too many times.

To avoid this we can abstract this as a dictionary. This way it provides better readability too.
